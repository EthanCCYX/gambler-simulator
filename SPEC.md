# 赌徒模拟器 - SPEC.md

## 1. Concept & Vision

**让抽象的概率论变成可触摸的痛感。** 一个精密的概率模拟器，用实时动画和冷酷的数据揭示赌局的真相：当玩家以为自己在"掌控节奏"时，数学正在悄悄清空他们的账户。视觉上像拉斯维加斯的监控室——黑色背景、绿色数字、冷峻的数据流，带有轻微的不安感。

核心体验：*从"我觉得我能赢"到"我亲眼看见我输光了"——中间只需要 200 局。*

## 4. Features & Interactions

### 4.1 两种游戏模式

#### 模式 A: 经典赌徒（Fair Game）
- 掷骰子（1-6），**大于 3 则赢**（胜率 50%）
- 赔率 1:1（本金不变，赢了翻倍）
- 目的：直观感受大数定律，50% 胜率长期必归零

#### 模式 B: 百家乐变体（The House Always Wins）
- 胜率 **48%（玩家）** vs **52%（庄家）**
- 庄家抽水：每次下注抽取 5%
- 目的：即使胜率接近 50%，庄家抽水也会导致必然亏损

### 4.2 下注策略模拟（可切换）

| 策略 | 行为描述 |
|------|---------|
| **固定注码** | 每局下注固定金额，观察稳定亏损 |
| **倍投法** | 输了翻倍下注，赢了回本（经典作死） |
| **本金比例** | 每次下注当前本金的 10%，感受"温水煮青蛙" |
| **随机金额** | 情绪化下注，有时All-in，有时只下一点 |
| **martingale-stop** | 倍投法 + 达到止损线强制停止 |

### 4.3 核心交互

**开始模拟**
- 点击"开始"后，后台以 50ms/局的速度自动运行
- 可调节速度：10ms（闪电）/ 100ms（标准）/ 500ms（慢速）
- 支持随时暂停/继续

**实时数据**
- 余额每帧更新，数字滚动动画
- 骰子翻转动画（CSS 3D transform）
- 每局结果在大字报区域闪现 800ms
- 亏损时屏幕边缘红色脉冲

**统计面板（右侧）**
- 实时余额曲线（Canvas，滚动窗口 200 局）
- 胜率直方图（最近 100 局的胜负分布）
- 最高连胜/连负记录
- 资金蒸发速度（每小时理论亏损额）

**危险警报**
- 余额 < 50% 时：橙色警告
- 余额 < 20% 时：红色警报 + 脉冲
- 余额 < 5% 时：最终警告 + 碎屏特效

**历史热力图**
- 最近 50 局结果用方格展示
- 赢=绿，输=红，连续赢/输加深颜色

**关键结论卡片**（动态更新）
- "你已经玩了 N 局，理论亏损 M%（基于赔率）"
- "按此速度，预计再玩 X 局将破产"
- "你的实际胜率是 Y%（对比理论值）"

### 4.4 教育性注释

每个面板旁边有 `(?)` 图标，悬停显示概率学解释：
- 赌徒谬误："连续输了5局，第6局该赢了吧？" → 错误，每局独立事件
- 大数定律："玩得越久，实际胜率越接近理论值"
- 庄家优势："即使胜率49%，长期来看也必然亏损"

## 6. Technical Approach

### 架构
- **单文件 HTML**（零依赖，便于分发和演示）
- 内联 CSS + JavaScript
- Canvas 2D 用于图表
- CSS 动画用于 UI 效果

### 核心算法

```javascript
// 每局模拟
function simulateRound(balance, bet, strategy) {
  const roll = Math.random(); // 0~1
  const WIN_PROB = 0.5; // 经典模式
  const HOUSE_EDGE = 0.05; // 百家乐抽水
  const threshold = 1/6 * 5; // >3 的概率

  const isWin = roll < threshold;
  const payout = isWin ? bet * (1 - HOUSE_EDGE) : -bet;

  return { isWin, payout, newBalance: balance + payout };
}
```

### 数据存储
- 模拟过程数据存储在内存数组（最多保留 10000 局）
- 每 100 局做一次快照，用于曲线图采样

### 性能
- Canvas 渲染使用 `requestAnimationFrame`
- DOM 更新使用批量 `requestAnimationFrame` 合并
- 历史记录使用虚拟滚动（只渲染可视区域）

## 7. Educational Goals

最终让用户亲身感受到：

1. **50% 胜率 ≠ 能赢** → 本金会无限趋于 0
2. **庄家抽水是慢性死亡** → 即使 49% 胜率，长期也归零
3. **倍投法必死** → 唯一区别是爆仓快还是慢
4. **赌徒谬误** → 连输不增加下一局胜率
5. **大数定律的冷酷** → 玩得越久，亏损越接近理论值
